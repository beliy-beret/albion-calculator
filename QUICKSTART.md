# Быстрый запуск 🚀

## Команды для запуска

```bash
# Установка зависимостей (только первый раз)
npm install

# Запуск в режиме разработки
npm run dev

# Или быстрый запуск с автооткрытием браузера
npm run start

# Сборка для продакшна
npm run build

# Предпросмотр сборки
npm run preview
```

## URL для доступа

После запуска приложение будет доступно по адресу:
**http://localhost:3000**

## Преимущества Vite

✅ **Мгновенный запуск** - холодный старт за ~300ms  
✅ **Горячая перезагрузка** - изменения применяются мгновенно  
✅ **Оптимизированная сборка** - быстрая сборка с Rollup  
✅ **ES модули** - нативная поддержка современного JS
