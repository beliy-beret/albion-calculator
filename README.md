# Albion Online - Калькулятор крафта дерева 🌲

Веб-приложение для расчета эффективности крафта древесных ресурсов в игре Albion Online.

## Возможности

- 📊 Расчет максимального количества брусьев из доступных ресурсов
- 🔄 Учет процента возврата ресурсов (15.2% - 45%)
- 📈 Отображение эффективности крафта
- 🎯 Поддержка всех тиров дерева (T2-T7)
- 💰 **Экономические расчеты**: себестоимость единицы продукции
- 📊 **Детальная смета**: стоимость материалов и крафта
- 📱 Адаптивный дизайн для мобильных устройств

## Технологии

- **React 18** - UI библиотека
- **Ant Design 5** - компоненты интерфейса
- **Vite** - быстрый сборщик и dev server

## Как использовать

1. **Выберите тир дерева** - от T2 (Береза) до T7 (Пепельная кора)
2. **Введите количество бревен** - сколько у вас логов для крафта
3. **Укажите количество брусьев предыдущего тира** - для T3+ рецептов
4. **Настройте процент возврата ресурсов**:
   - Базовый: 15.2%
   - С фокусом: до 30%
   - В хайдаутах с бонусами: до 45%
5. **💰 Введите экономические параметры**:
   - Цена бревна за единицу (из маркетплейса)
   - Цена бруска предыдущего тира
   - Стоимость крафта за единицу (налоги станка)
6. **Нажмите "Рассчитать"** для получения результатов

## Рецепты крафта

| Тир | Название | Рецепт |
|-----|----------|--------|
| T2 | Birch Plank (Березовый брусок) | 1 Birch Log |
| T3 | Chestnut Plank (Каштановый брусок) | 2 Chestnut Logs + 1 Birch Plank |
| T4 | Pine Plank (Сосновый брусок) | 2 Pine Logs + 1 Chestnut Plank |
| T5 | Cedar Plank (Кедровый брусок) | 3 Cedar Logs + 1 Pine Plank |
| T6 | Bloodoak Plank (Брусок кровавого дуба) | 4 Bloodoak Logs + 1 Cedar Plank |
| T7 | Ashenbark Plank (Брусок пепельной коры) | 5 Ashenbark Logs + 1 Bloodoak Plank |

## Результаты расчета

Калькулятор показывает:

- **Максимальное количество брусьев** для крафта
- **Эффективность** использования ресурсов
- **💰 Себестоимость единицы** готового продукта
- **Использованные ресурсы** с учетом возврата
- **Остатки ресурсов** после крафта
- **Возвращенные ресурсы** благодаря проценту возврата
- **Детальная смета**: стоимость бревен, брусков и крафта
- **Общая стоимость** производства

## Полезные советы

1. **Фокус** значительно увеличивает возврат ресурсов
2. **Хайдауты** дают лучший возврат, чем городские станки
3. **Местные бонусы** городов влияют на эффективность крафта
4. **Мебель** - хороший выбор для начинающих крафтеров
