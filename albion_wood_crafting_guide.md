# Гайд по крафту дерева в Albion Online

## Основы системы

### Инструменты для сбора
- **Лесопильный топор** — основной инструмент для сбора дерева
- Топор уровня T(X) позволяет собирать дерево уровня T(X) и T(X+1)

### Станки для переработки
- **Лесопилка (Lumbermill)** — станок для переработки логов в доски
- Расположены в лесных районах городов

## Рецепты переработки на Лесопилке

### Создание досок из логов:

| Уровень досок | Необходимые ресурсы |
|---------------|-------------------|
| T2 Birch Plank | 1 Birch Log |
| T3 Chestnut Plank | 2 Chestnut Logs + 1 Birch Plank |
| T4 Pine Plank | 2 Pine Logs + 1 Chestnut Plank |
| T5 Cedar Plank | 3 Cedar Logs + 1 Pine Plank |
| T6 Blood Oak Plank | 4 Blood Oak Logs + 1 Cedar Plank |
| T7 Ashen Bark Plank | 5 Ashen Bark Logs + 1 Blood Oak Plank |

## Локации сбора

### Безопасные зоны (синие)
- T1-T4 дерево
- Безопасно для новичков

### Желтые зоны  
- T3-T5 дерево
- Небольшой риск PvP

### Красные зоны
- T4-T8 дерево
- Высокий риск PvP
- Можно потерять все снаряжение

### Черные зоны
- T5-T8 дерево
- Максимальный риск
- Наилучшие ресурсы

### Примеры T8 локаций:
- **Eye of the Forest** (красная зона):
  - Bloodoak Logs (T6): 200 узлов
  - Ashenbark Logs (T7): 70 узлов  
  - Whitewood Logs (T8): 60 узлов
- **Redtree Enclave** (черная зона)

## Экономические аспекты

### Возврат ресурсов (Resource Return Rate)
- Ключевой показатель прибыли крафтера
- В Убежищах (Hideouts): до 45% возврата
- Влияет на себестоимость производства

### Местные бонусы
- Каждый город имеет бонусы к определенным предметам из дерева
- Станки в городах взимают налог
- Рекомендуется договариваться с владельцами о сниженных ставках

### Фокус (Focus)
- Используется для увеличения эффективности крафта
- Позволяет получать больше ресурсов при переработке

## Рекомендации для новичков

### Выгодные направления:
1. **Мебель** (сундуки, столы, кровати)
   - Нет уровней мастерства (Specs)
   - Фиксированная стоимость фокуса  
   - Стабильный доход

2. **Дерево как ресурс**
   - Один из самых доступных по цене
   - Дешевле кожи и волокна
   - Хорошая точка входа в крафт

### Советы по безопасности:
- Начинайте сбор в безопасных зонах
- Изучите карту перед походом в опасные зоны
- Имейте финансовый запас на случай потери снаряжения
- Используйте дешевое снаряжение для сбора в PvP зонах

## Инструменты для расчетов

При создании калькулятора учитывайте:
- Стоимость ресурсов на рынке
- Местные налоги и бонусы
- Процент возврата ресурсов
- Затраты фокуса
- Время на сбор и переработку
